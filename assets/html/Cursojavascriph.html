Tema 1: Introducción a JavaScript

JavaScript es un lenguaje de programación que permite crear contenido dinámico en los sitios web.

Ejemplo práctico:

                            console.log("Hola, Mundo!");

Tema 2: Variables y tipos de datos

Las variables son contenedores para almacenar datos. Los tipos de datos comunes en JavaScript incluyen números, cadenas y booleanos.

Ejemplo práctico:

                                    let nombre = "Juan";
                               let edad = 30;
                               let esEstudiante = true;

Tema 3: Operadores

Los operadores se utilizan para realizar operaciones sobre variables y valores. Incluyen operadores aritméticos, de asignación y de comparación.

Ejemplo práctico:

                                    let suma = 5 + 3; // 8
                            let esIgual = (5 == 5); // true

Tema 4: Estructuras de control

Las estructuras de control, como if, else y switch, permiten tomar decisiones basadas en condiciones.

Ejemplo práctico:

                                    let edad = 18;
                if (edad >= 18) {
                                   console.log("Eres adulto.");
                    } else {
                            console.log("Eres menor de edad.");
                }

Tema 5: Funciones

Las funciones son bloques de código diseñados para realizar una tarea particular, y pueden ser reutilizadas en cualquier parte del programa.

Ejemplo práctico:

                function saludar(nombre) {
                         console.log("Hola, " + nombre);
                     }
                        saludar("Ana"); // Hola, Ana

Tema 6: Objetos y Arrays

Los objetos y arrays son estructuras de datos que permiten almacenar múltiples valores en una sola variable.

Ejemplo práctico:

let persona = {
    nombre: "Carlos",
    edad: 25,
    esEstudiante: true
};
let frutas = ["Manzana", "Banana", "Cereza"];

Tema 7: Manipulación del DOM

El DOM (Document Object Model) es una interfaz que permite a los programas y scripts acceder y actualizar el contenido, la estructura y el estilo de los documentos web.

Ejemplo práctico:

            document.getElementById("miElemento").innerHTML = "Nuevo contenido";

Tema 8: Eventos

Los eventos en JavaScript permiten ejecutar código en respuesta a acciones del usuario, como hacer clic en un botón o mover el ratón.

Ejemplo práctico:

                                    document.getElementById("miBoton").addEventListener("click", function() {
    alert("Botón clicado");
});

Tema 9: Asincronía y Promesas

La programación asíncrona en JavaScript permite que el código se ejecute sin bloquear el hilo principal. Las promesas son una forma de manejar operaciones asíncronas.

Ejemplo práctico:

fetch('https://api.example.com/datos')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Tema 10: Clases y Prototipos

Las clases y prototipos en JavaScript permiten crear objetos y reutilizar el código de manera eficiente.

Ejemplo práctico:

class Persona {
    constructor(nombre, edad) {
        this.nombre = nombre;
        this.edad = edad;
    }
    saludar() {
        console.log(`Hola, me llamo ${this.nombre}`);
    }
}
let persona1 = new Persona("Lucía", 28);
persona1.saludar(); // Hola, me llamo Lucía

Tema 11: Módulos

Los módulos en JavaScript permiten dividir el código en archivos separados, facilitando la organización y mantenimiento del código.

Ejemplo práctico:

// archivo.js
export function saludar(nombre) {
    return `Hola, ${nombre}`;
}
// main.js
import { saludar } from './archivo.js';
console.log(saludar('Mundo')); // Hola, Mundo

Tema 12: APIs del navegador

Las APIs del navegador, como la API de Geolocalización, la API de Almacenamiento y la API de Canvas, permiten acceder a características avanzadas del navegador.

Ejemplo práctico:

                                    navigator.geolocation.getCurrentPosition(function(position) {
    console.log("Latitud: " + position.coords.latitude);
    console.log("Longitud: " + position.coords.longitude);
});
