Tema 1: Introducción a HTML

HTML es el lenguaje de marcado estándar para crear páginas web. Permite estructurar el contenido en la web.

Ejemplo práctico:

                                    <!DOCTYPE html>
                                    <html>
                                    <head>
                      <title>Página de ejemplo</title>
                                    </head>
                                     <body>
                             <h1>Hola, mundo!</h1>
                      <p>Este es un párrafo en HTML.</p>
                                    </body>
                                   </html>

Tema 2: Estructura básica de un documento HTML

La estructura básica de un documento HTML incluye las etiquetas <!DOCTYPE html>, <html>, <head>, y <body>.

Ejemplo práctico:

                        <!DOCTYPE html>
                            <html>
                            <head>
                            <title>Mi Página</title>
                            </head>
                            <body>
                            <h1>Bienvenido a mi página</h1>
                            </body>
                            </html>

Tema 3: Etiquetas de encabezado

Las etiquetas de encabezado, <h1> a <h6>, se utilizan para definir los títulos y subtítulos en una página web.

Ejemplo práctico:

                                 <h1>Encabezado 1</h1>
                                    <h2>Encabezado 2</h2>
                                    <h3>Encabezado 3</h3>

Tema 4: Párrafos y texto

Las etiquetas <p> se utilizan para definir párrafos, y <strong> o <em> se utilizan para texto en negrita o cursiva, respectivamente.

Ejemplo práctico:

                        <p>Este es un párrafo.</p>
<p>Este es un <strong>texto en negrita</strong> y este es un <em>texto en cursiva</em>.</p>

Tema 5: Listas

HTML soporta listas ordenadas (<ol>) y listas no ordenadas (<ul>).

Ejemplo práctico:

                    <ul>
                        <li>Elemento de lista no ordenada</li>
                        <li>Otro elemento</li>
                </ul>
                 <ol>
                        <li>Elemento de lista ordenada</li>
                        <li>Otro elemento</li>
                </ol>

Tema 6: Enlaces

Los enlaces se crean utilizando la etiqueta <a>, y el atributo href define la URL del destino del enlace.

Ejemplo práctico:

                <a href="https://www.ejemplo.com">Visita Ejemplo.com</a>

Tema 7: Imágenes

Las imágenes se insertan usando la etiqueta <img>, con atributos como src para la fuente de la imagen y alt para el texto alternativo.

Ejemplo práctico:

                <img src="imagen.jpg" alt="Descripción de la imagen">

Tema 8: Tablas

Las tablas en HTML se crean usando las etiquetas <table>, <tr>, <td> y <th>.

Ejemplo práctico:

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

Tema 9: Formularios

Los formularios permiten a los usuarios enviar datos a un servidor. Se crean usando la etiqueta <form> y varios elementos de entrada como <input>.

Ejemplo práctico:

<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

Tema 10: Audio y Video

HTML5 introdujo las etiquetas <audio> y <video> para la reproducción de archivos de audio y video directamente en la página web.

Ejemplo práctico:

 <audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Tu navegador no soporta la etiqueta de audio.
</audio>

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Tu navegador no soporta la etiqueta de video.
</video>

Tema 11: Elementos de bloque y en línea

Los elementos de bloque ocupan todo el ancho disponible y siempre comienzan en una nueva línea, mientras que los elementos en línea solo ocupan el espacio necesario y no forzan un salto de línea.

Ejemplo práctico:

            <div>Este es un div, que es un elemento de bloque.</div>
        <span>Este es un span, que es un elemento en línea.</span>

Tema 12: Comentarios en HTML

Los comentarios en HTML se escriben entre <!-- --> y no se muestran en el navegador.

Ejemplo práctico:

                        <!-- Este es un comentario en HTML -->
