Tema 1: Introducción a C#

C# es un lenguaje de programación desarrollado por Microsoft, ampliamente utilizado en aplicaciones .NET, desde aplicaciones de escritorio hasta servicios web.

Ejemplo práctico:

                                    using System;
            class Program {
                    static void Main() {
                             Console.WriteLine("Hola, Mundo!");
                          }
}

Tema 2: Variables y tipos de datos

Las variables en C# son contenedores para almacenar datos. Los tipos de datos incluyen enteros, flotantes, caracteres y booleanos.

Ejemplo práctico:

                                      int edad = 25;
                                float altura = 1.75f;
                                char inicial = 'J';
                                bool esEstudiante = true;

Tema 3: Operadores

Los operadores se utilizan para realizar operaciones sobre variables y valores, incluyendo operadores aritméticos, de asignación y de comparación.

Ejemplo práctico:

                                    int suma = 5 + 3; // 8
                            bool esIgual = (5 == 5); // true

Tema 4: Estructuras de control

Las estructuras de control, como if, else y switch, permiten tomar decisiones basadas en condiciones.

Ejemplo práctico:

                                    int edad = 18;
if (edad >= 18) {
    Console.WriteLine("Eres adulto.");
} else {
    Console.WriteLine("Eres menor de edad.");
}

Tema 5: Funciones

Las funciones en C# permiten encapsular bloques de código para realizar tareas específicas, y pueden ser reutilizadas en cualquier parte del programa.

Ejemplo práctico:

                                    void Saludar(string nombre) {
                             Console.WriteLine("Hola, " + nombre);
                        }
                        static void Main() {
                        Saludar("Ana"); // Hola, Ana
                                    }

Tema 6: Arreglos y colecciones

Los arreglos y las colecciones son formas de almacenar múltiples valores en una sola variable. Las colecciones proporcionan una mayor flexibilidad que los arreglos.

Ejemplo práctico:

                                    int[] numeros = {1, 2, 3, 4, 5};

                          List<string> nombres = new List<string>();

                            nombres.Add("Carlos");
                            
                            nombres.Add("Ana");

Tema 7: Referencias y punteros

En C#, los punteros son más limitados y generalmente se utilizan en contextos de interoperabilidad o programación no segura. Las referencias permiten trabajar con objetos de manera eficiente.

Ejemplo práctico:

                                    int valor = 10;

                                 int referencia = valor;

                                referencia = 20;

                 Console.WriteLine(valor); // 10 (valor no cambia)

Tema 8: Clases y objetos

Las clases y objetos son los bloques de construcción fundamentales de la programación orientada a objetos en C#.

Ejemplo práctico:

                                    class Persona {
    public string Nombre { get; set; }
    public int Edad { get; set; }

    public void Saludar() {
        Console.WriteLine("Hola, me llamo " + Nombre);
    }
}

static void Main() {
    Persona persona1 = new Persona();
    persona1.Nombre = "Lucía";
    persona1.Edad = 28;
    persona1.Saludar(); // Hola, me llamo Lucía
}

Tema 9: Herencia y polimorfismo

La herencia permite crear nuevas clases basadas en clases existentes. El polimorfismo permite utilizar una interfaz común para diferentes tipos de datos.

Ejemplo práctico:

                                    class Animal {
    public virtual void HacerSonido() {
        Console.WriteLine("Sonido de animal");
    }
}

class Perro : Animal {
    public override void HacerSonido() {
        Console.WriteLine("Guau");
    }
}

static void Main() {
    Animal miAnimal = new Perro();
    miAnimal.HacerSonido(); // Guau
}

Tema 10: Manejo de archivos

El manejo de archivos en C# permite leer y escribir datos en archivos, facilitando el almacenamiento y recuperación de información.

Ejemplo práctico:

                                    using System.IO;

static void Main() {
    File.WriteAllText("ejemplo.txt", "Hola, archivo!");
    string contenido = File.ReadAllText("ejemplo.txt");
    Console.WriteLine(contenido);
}

Tema 11: Expresiones lambda

Las expresiones lambda permiten definir funciones anónimas que pueden ser utilizadas en operaciones como LINQ.

Ejemplo práctico:

                                    Func<int, int> cuadrado = x => x * x;
Console.WriteLine(cuadrado(5)); // 25

Tema 12: Librerías y namespaces

Las librerías en C# proporcionan clases y métodos que facilitan el desarrollo de aplicaciones. Los namespaces permiten organizar y agrupar estas librerías.

Ejemplo práctico:

                                    using System;
using System.Collections.Generic;

static void Main() {
    List<int> numeros = new List<int>{1, 2, 3, 4, 5};
    numeros.Sort();
    foreach (int num in numeros) {
        Console.Write(num + " ");
    } // 1 2 3 4 5
}
