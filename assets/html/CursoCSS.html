Explora cómo CSS te permite diseñar y estilizar páginas web de manera atractiva y coherente.

Tema 1: Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML.

Ejemplo práctico:

body {
    background-color: lightblue;
}
h1 {
    color: navy;
    margin-left: 20px;
}

Tema 2: Selectores

Los selectores se utilizan para seleccionar los elementos HTML que queremos estilizar.

Ejemplo práctico:

                                    /* Selector de tipo */
p {
    color: red;
}

/* Selector de clase */
.intro {
    font-size: 20px;
}

/* Selector de ID */
#header {
    background-color: yellow;
}

Tema 3: Propiedades y valores

Las propiedades son los aspectos de los elementos que se pueden cambiar, y los valores son las especificaciones de esas propiedades.

Ejemplo práctico:

h1 {
    font-family: Arial, sans-serif;
    font-size: 24px;
    color: blue;
}

Tema 4: Colores y fondos

CSS permite cambiar los colores del texto y de fondo de los elementos HTML.

Ejemplo práctico:

body {
    background-color: #f0f0f0;
}
h1 {
    color: #333;
}
p {
    background-color: yellow;
    color: red;
}

Tema 5: Fuentes y texto

CSS se utiliza para cambiar la fuente, el tamaño del texto y otros aspectos tipográficos.

Ejemplo práctico:

p {
    font-family: "Times New Roman", Times, serif;
    font-size: 16px;
    font-weight: bold;
    text-align: center;
}

Tema 6: Modelo de caja (Box Model)

El modelo de caja de CSS describe cómo se colocan los elementos en la página, incluyendo márgenes, bordes, rellenos y el contenido real.

Ejemplo práctico:

div {
    width: 300px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}

Tema 7: Bordes

Se pueden añadir bordes a los elementos HTML y estilizar su grosor, color y estilo.

Ejemplo práctico:

p {
    border: 1px solid black;
    border-radius: 5px;
}

Tema 8: Margen y relleno (Margin & Padding)

Margin y padding son propiedades que permiten crear espacio alrededor de los elementos.

Ejemplo práctico:

div {
    margin: 20px;
    padding: 10px;
}

Tema 9: Flexbox

Flexbox es un modelo de diseño que proporciona una forma más eficiente de distribuir el espacio entre los elementos de un contenedor, incluso cuando su tamaño es desconocido y/o dinámico.

Ejemplo práctico:

div.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Tema 10: Posicionamiento

El posicionamiento en CSS permite especificar la posición de un elemento en la página. Los valores comunes incluyen static, relative, absolute y fixed.

Ejemplo práctico:

div {
    position: absolute;
    top: 50px;
    left: 100px;
}

Tema 11: Diseño de cuadrícula (Grid)

El diseño de cuadrícula (CSS Grid) es una técnica poderosa para crear layouts complejos con filas y columnas.

Ejemplo práctico:

div.container {
    display: grid;
    grid-template-columns: auto auto auto;
    gap: 10px;
}
div.item {
    background-color: lightblue;
    padding: 20px;
}

Tema 12: Medios y consultas de medios (Media Queries)

Las consultas de medios (media queries) permiten aplicar estilos CSS según las características del dispositivo, como el ancho de la pantalla, para hacer diseños responsivos.

Ejemplo práctico:

@media only screen and (max-width: 600px) {
    body {
        background-color: lightgreen;
    }
}
