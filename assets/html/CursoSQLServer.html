Explora cómo SQL Server te permite gestionar y manipular bases de datos de manera eficiente.

Tema 1: Introducción a SQL Server

SQL Server es un sistema de gestión de bases de datos relacional desarrollado por Microsoft, que utiliza el lenguaje SQL para gestionar datos.

Ejemplo práctico:

                SELECT GETDATE(); -- Muestra la fecha y hora actual

Tema 2: Creación de bases de datos

Para crear una nueva base de datos en SQL Server, utilizamos el comando `CREATE DATABASE`.

Ejemplo práctico:

                        CREATE DATABASE MiBaseDeDatos;

Tema 3: Creación y modificación de tablas

Las tablas se crean y modifican usando los comandos `CREATE TABLE` y `ALTER TABLE` para definir las estructuras y campos de los datos.

Ejemplo práctico:

CREATE TABLE Empleados (
    ID INT PRIMARY KEY,
    Nombre NVARCHAR(50),
    FechaContratacion DATE
);

ALTER TABLE Empleados
ADD Salario DECIMAL(10, 2);

Tema 4: Consultas básicas

Las consultas básicas se realizan con el comando `SELECT`, permitiendo recuperar datos de una o más tablas.

Ejemplo práctico:

                            SELECT Nombre, FechaContratacion
                               FROM Empleados
                               WHERE Salario > 50000;

Tema 5: Inserción, actualización y eliminación de datos

Los datos en las tablas se insertan con `INSERT INTO`, se actualizan con `UPDATE` y se eliminan con `DELETE`.

Ejemplo práctico:

INSERT INTO Empleados (ID, Nombre, FechaContratacion, Salario)
VALUES (1, 'Juan Pérez', '2024-01-15', 60000);

UPDATE Empleados
SET Salario = 65000
WHERE ID = 1;

DELETE FROM Empleados
WHERE ID = 1;

Tema 6: Funciones y procedimientos almacenados

Las funciones y procedimientos almacenados permiten encapsular lógica compleja para reutilizarla y mejorar el rendimiento.

Ejemplo práctico:

CREATE PROCEDURE ObtenerEmpleados
AS
BEGIN
    SELECT * FROM Empleados;
END;

CREATE FUNCTION CalcularBonificacion (@Salario DECIMAL(10, 2))
RETURNS DECIMAL(10, 2)
AS
BEGIN
    RETURN @Salario * 0.10;
END;

Tema 7: Índices y claves foráneas

Los índices mejoran el rendimiento de las consultas, mientras que las claves foráneas aseguran la integridad referencial entre tablas.

Ejemplo práctico:

CREATE INDEX idx_Nombre
ON Empleados (Nombre);

ALTER TABLE Empleados
ADD CONSTRAINT fk_Departamento
FOREIGN KEY (DepartamentoID) REFERENCES Departamentos(ID);

Tema 8: Consultas avanzadas

Las consultas avanzadas incluyen el uso de `JOIN`, subconsultas y funciones agregadas para obtener datos complejos y resumidos.

Ejemplo práctico:

                  SELECT e.Nombre, d.Nombre AS Departamento
                FROM Empleados e
                JOIN Departamentos d ON e.DepartamentoID = d.ID
                WHERE e.Salario > 50000;

Tema 9: Seguridad y permisos

La seguridad en SQL Server se gestiona mediante roles y permisos para controlar el acceso y la modificación de los datos.

Ejemplo práctico:

                    CREATE LOGIN UsuarioAdmin WITH PASSWORD = 'ContraseñaSegura!';
CREATE USER UsuarioAdmin FOR LOGIN UsuarioAdmin;
ALTER ROLE db_owner ADD MEMBER UsuarioAdmin;

Tema 10: Mantenimiento de bases de datos

El mantenimiento incluye tareas como la optimización de consultas, el respaldo de datos y la restauración de bases de datos.

Ejemplo práctico:

                              BACKUP DATABASE MiBaseDeDatos
TO DISK = 'C:\\Backup\\MiBaseDeDatos.bak';

RESTORE DATABASE MiBaseDeDatos
FROM DISK = 'C:\\Backup\\MiBaseDeDatos.bak';
