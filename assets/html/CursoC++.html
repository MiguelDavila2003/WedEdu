Tema 1: Introducción a C++

C++ es un lenguaje de programación de propósito general que se utiliza en una variedad de aplicaciones, desde sistemas operativos hasta videojuegos.

Ejemplo práctico:

                                    #include <iostream>
                                        using namespace std;
                                        
                                        int main() {
                                            cout << "Hola, Mundo!" << endl;
                                            return 0;
                                        }

Tema 2: Variables y tipos de datos

Las variables son contenedores para almacenar datos. Los tipos de datos comunes en C++ incluyen enteros, flotantes, caracteres y booleanos.

Ejemplo práctico:

                                    int edad = 25;
                            float altura = 1.75;
                            char inicial = 'J';
                            bool esEstudiante = true;

Tema 3: Operadores

Los operadores se utilizan para realizar operaciones sobre variables y valores. Incluyen operadores aritméticos, de asignación y de comparación.

Ejemplo práctico:

                                    int suma = 5 + 3; // 8
                            bool esIgual = (5 == 5); // true

Tema 4: Estructuras de control

Las estructuras de control, como if, else y switch, permiten tomar decisiones basadas en condiciones.

Ejemplo práctico:

int edad = 18;
if (edad >= 18) {
    cout << "Eres adulto." << endl;
} else {
    cout << "Eres menor de edad." << endl;
}

Tema 5: Funciones

Las funciones son bloques de código diseñados para realizar una tarea particular, y pueden ser reutilizadas en cualquier parte del programa.

Ejemplo práctico:

void saludar(string nombre) {
    cout << "Hola, " << nombre << endl;
}

int main() {
    saludar("Ana"); // Hola, Ana
    return 0;
}

Tema 6: Arreglos y estructuras

Los arreglos y las estructuras son formas de almacenar múltiples valores en una sola variable. Las estructuras permiten agrupar diferentes tipos de datos.

Ejemplo práctico:

int numeros[] = {1, 2, 3, 4, 5};

struct Persona {
    string nombre;
    int edad;
};

Persona persona1;
persona1.nombre = "Carlos";
persona1.edad = 25;

Tema 7: Punteros

Los punteros son variables que almacenan la dirección de memoria de otra variable, lo que permite manipular datos de forma eficiente.

Ejemplo práctico:

                                    int valor = 10;
                            int* ptr = &valor;
                 cout << "Valor: " << *ptr << endl; // 10

Tema 8: Clases y objetos

Las clases y objetos son los bloques de construcción fundamentales de la programación orientada a objetos en C++.

Ejemplo práctico:

class Persona {
public:
    string nombre;
    int edad;

    void saludar() {
        cout << "Hola, me llamo " << nombre << endl;
    }
};

int main() {
    Persona persona1;
    persona1.nombre = "Lucía";
    persona1.edad = 28;
    persona1.saludar(); // Hola, me llamo Lucía
    return 0;
}

Tema 9: Herencia y polimorfismo

La herencia permite crear nuevas clases basadas en clases existentes. El polimorfismo permite utilizar una interfaz común para diferentes tipos de datos.

Ejemplo práctico:

class Animal {
public:
    virtual void hacerSonido() {
        cout << "Sonido de animal" << endl;
    }
};

class Perro : public Animal {
public:
    void hacerSonido() override {
        cout << "Guau" << endl;
    }
};

int main() {
    Animal* miAnimal = new Perro();
    miAnimal->hacerSonido(); // Guau
    delete miAnimal;
    return 0;
}

Tema 10: Manejo de archivos

El manejo de archivos en C++ permite leer y escribir datos en archivos, facilitando el almacenamiento y recuperación de información.

Ejemplo práctico:

#include <fstream>

int main() {
    ofstream archivo("ejemplo.txt");
    archivo << "Hola, archivo!" << endl;
    archivo.close();

    ifstream leerArchivo("ejemplo.txt");
    string linea;
    while (getline(leerArchivo, linea)) {
        cout << linea << endl;
    }
    leerArchivo.close();
    return 0;
}

Tema 11: Plantillas (Templates)

Las plantillas permiten crear funciones y clases genéricas, facilitando el uso de código reutilizable para diferentes tipos de datos.

Ejemplo práctico:

template <typename T>
T obtenerMaximo(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    cout << obtenerMaximo(10, 20) << endl; // 20
    cout << obtenerMaximo(5.5, 2.3) << endl; // 5.5
    return 0;
}

Tema 12: Librerías estándar

La Standard Template Library (STL) proporciona una colección de clases y funciones que facilitan la manipulación de datos y estructuras.

Ejemplo práctico:

#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> numeros = {5, 3, 8, 1, 2};
    sort(numeros.begin(), numeros.end());

    for (int num : numeros) {
        cout << num << " ";
    } // 1 2 3 5 8
    return 0;
}
